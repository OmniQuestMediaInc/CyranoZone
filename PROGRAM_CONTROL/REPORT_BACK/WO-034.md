## Task Summary

### TASK 1 — Suppression Registry Schema ✅

**File:** `infra/postgres/init-ledger.sql`

- Created `content_hash_registry` table with:
  - `hash CHAR(64) PRIMARY KEY` — SHA-256 hex string
  - `case_id UUID NOT NULL` — Reference to Incident Object
  - `action_taken VARCHAR(20) CHECK IN ('SUPPRESS', 'BLOCK_REUPLOAD')`
  - `rule_applied_id VARCHAR(100)` — Reference to Corpus v10 Section 5
  - `created_at_utc TIMESTAMPTZ NOT NULL DEFAULT NOW()` — UTC timestamp
  - `created_at_toronto TEXT NOT NULL` — America/Toronto local time (auto-computed by BEFORE INSERT trigger `trg_content_hash_registry_set_toronto_ts`)
- Tamper-evident: `trg_content_hash_registry_no_update` and `trg_content_hash_registry_no_delete` raise EXCEPTION on any mutation attempt
- Append-only: INSERT ONLY per OQMI Doctrine

### TASK 2 — ProvisionalSuppressionService ✅

**File:** `services/core-api/src/safety/provisional-suppression.service.ts`

- `suppressContent(contentId, caseId, ruleAppliedId)`: Sets PROVISIONAL suppression, emits SEV1 Audit_Event
- `liftSuppression(contentId, moderatorId)`: Lifts PROVISIONAL suppression (reversible); throws if FINALIZED
- `finalizeAndRegisterHash(contentId, contentHash, action)`: Writes hash to registry after human confirmation
- `generateEvidencePacket(caseId)`: Collects incident timeline, hash references, rule applied (Task 4)

### TASK 3 — Upload Pipeline Interceptor ✅

**File:** `services/core-api/src/safety/upload-interceptor.middleware.ts`

- `intercept(candidate: UploadCandidate)`: Hashes file buffer with SHA-256, checks `content_hash_registry`
- If match found: blocks upload, logs SEV1 incident, returns `allowed: false`
- If no match: allows upload, returns `allowed: true`
- `static computeHash(buffer: Buffer)`: SHA-256 utility

### TASK 4 — Evidence Packet Stub ✅

**Implemented in:** `ProvisionalSuppressionService.generateEvidencePacket(caseId)`

- Collects all `content_hash_registry` entries for the case
- Includes provisional suppression events in the timeline
- Returns `hash_reference` (not file content), `rule_applied_id`, `incident_timeline`, `generated_at_utc`
