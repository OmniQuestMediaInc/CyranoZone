# NCII Takedown Log — Evidence Template

**WO: WO-INIT-001**

## Purpose

This log records Non-Consensual Intimate Image (NCII) takedown requests and their resolution status in compliance with applicable law and OQMI Content Safety Policy.

## Instructions

- Every NCII takedown request must be logged here as a new row in the table below.
- Entries are **append-only**. Do not modify or delete existing rows.
- Include hash/fingerprint of reported content where available.

## Status Values

| Status       | Meaning                                 |
| ------------ | --------------------------------------- |
| `OPEN`       | Received, pending review                |
| `IN_REVIEW`  | Under investigation                     |
| `TAKEN_DOWN` | Content removed from platform           |
| `ESCALATED`  | Referred to law enforcement or legal    |
| `REJECTED`   | Report found to be invalid after review |
| `CLOSED`     | Resolved and archived                   |
