# REFERENCE_LIBRARY/TEMPLATES

This directory contains reusable master templates for OQMInc workflows.
All templates use {PLACEHOLDER} substitution — fill in parameters
before use. See each template file for its parameter list.

## Templates in This Directory

| File                                        | Purpose                                                  |
| ------------------------------------------- | -------------------------------------------------------- |
| OQMInc_REFERENCE_LIBRARY_MASTER_TEMPLATE.md | Initialize Reference Library workflow in any OQMInc repo |
| OQMInc_OSS_HARVEST_TEMPLATE.md              | Harvest OSS repos into refs/\* reference branches        |
| OQMInc_DIRECTIVE_TEMPLATE.md                | Standard format for all CNZ directives                   |

## How Templates Are Maintained

Primary copy: Google Drive → BUILDD CONTROL - CNZ → OQMInc Templates
Repo copy: This directory (backup and agent-accessible version)

When a template is updated, update both locations.
Version history is tracked in each template file.
